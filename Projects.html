Project 4.  Rendering 3-D Animations of the Continuous Profile®


For the graphics artists out there, a fun project is to create a computer generated rendering of the Continuous Profile®.  All you need to do is capture the 2-D Facial Profile (see Project 1 above) and create a vector path for processing in any one of a number of rendering programs.  The resulting picture of your Continuous Profile® can be posted to your FaceBook page as an abstraction of your visage.  Or, you can create a looping video for continuous display in a browser.

Project 2.   Designing the Continuous Profile® Toolpath


Once the 2-D picture is taken, you can use your favorite Image editing software to make adjustments, including alignment and sizing.  Get creative in designing a base for the finished product when turned on a wood lathe or potters wheel.

Project 3.   Turning the Continuous Profile®


Now that you have created a toolpath / pattern, you can get down to business and create the physical object.  An experienced craftsman will likely opt to do this manually on a wood lathe or potters wheel.  In order to preserve the nuanced detail of the facial profile, I opted to use a duplicator mounted on a wood midi-lathe.  The act of turning creates an abstraction of the facial profile, often difficult to resolve back to the owner of the profile.

Project 5.  Automating the manufacture via Computer Numerical Control (CNC)


This new project requires the creation of a vector path (see Project 4) stored in DXF format.  In early tests, the DXF has successfully created a tool path in BobCAD and FlashCut CNC software.  Work is in-process to generate viable G-Code in order to program a small CNC lathe.

The Continuous Profile® is a multifaceted project, touching on many different technologies.  included are processes for capturing the 2-D facial profile; creating the design (tool path); turning the object on a wood lathe, creating the computer 3-D rendering.   A recent project has begun to explore the use of Computer Numerical Control  (CNC) to manufacture an object.
