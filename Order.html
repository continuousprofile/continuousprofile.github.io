3D Print:  The Continuous Profile provides guidance for creating 3D renderings in the .dae file format.  Using a SketchUp plugin, the .dae file can be Exported to .stl format, which is the standard format for most 3D printers.  This simple process has been used successfully to 3D print a Continuous Profile on a 3D Systems Cubify 3D printer.
